About BreachWatch

The Problem

The NIST NVD website is dense and difficult to navigate. BreachWatch presents the same data in a clean, searchable interface with severity scoring and bookmarking.

Who It's For

  • Developers checking if a dependency has known security issues
  • IT administrators without enterprise security tools
  • Cybersecurity students learning vulnerability management

Data Source

All data comes from the NIST National Vulnerability Database via their free public REST API.


https://services.nvd.nist.gov/rest/json/cves/2.0

Tech Stack

  • Node.js + Express
  • Supabase (saved CVEs database)
  • Chart.js (severity breakdown chart)
  • Swiper.js (search results slider)
  • NIST NVD API
  • HTML, CSS, JavaScript

API Endpoints

  • GET /api/critical - recent critical CVEs
  • GET /api/search?q=keyword - keyword search
  • GET /api/search?cveId=id - CVE ID lookup
  • GET /api/saved - retrieve watchlist
  • POST /api/saved - save a CVE
  • DELETE /api/saved/:cveId - remove a CVE
